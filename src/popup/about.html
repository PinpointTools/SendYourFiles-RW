The SYF Project

The SYF (Send Your Files) free and open source project is a hate comment towards Discord for their 10mb upload limit.

We originally wanted to make a app where it would Send Your Files through 3 different hosting services that we support. Originally used to be made with Python, requests and PyWebView. But after some testing with Linux and rethinking that we should probably add support for mobile, we decided to drop it for (p)npm, vanilla HTML and express.

We're open source. Which means, if you wanted to help us with our project by contributing or just finding a way to help us out or learn our code, you're always welcome to! We're never making our code closed-sourced.

We heavily rely on the file hosting. Without it, this project would not be possible. Thank you to Catbox, Litterbox and Buzzheavier! We love you.
